========================================================================
    CONSOLE APPLICATION : SWGModelExporter Project Overview
========================================================================

Tool reads StarWars Galaxies game resource database and extracts mesh (static and animated)
into FBX format for further processing.

Requires DirectXTex, ZLIB, Boost library (1.57 and later) and Autodesk FBX SDK 2016.2.
Third-party dependencies not included

DirectXTex lib can be acquired at https://github.com/Microsoft/DirectXTex/
ZLIB - http://www.zlib.net
Boost - http://boost.org
Autodesk FBX SDK - http://usa.autodesk.com/adsk/servlet/pc/item?siteID=123112&id=10775847 (or use search on Autodesk site)
